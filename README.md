# Melodix
Melodix is a personal project inspired by platforms like Spotify and Apple Music. It’s designed to explore how users can stream, organize, and enjoy music in a modern, intuitive way. The goal is to combine clean design with powerful features — playlists, recommendations, and a smooth listening experience — while experimenting with new ideas.

✨ What makes Melodix special:

Built with Spring Boot on the backend for speed, security, and scalability.
A sleek React frontend delivering a smooth and dynamic user experience.
Core features: playlists, smart recommendations, and seamless listening.
A creative playground to experiment with new ideas and innovative features.

🚀 Why it’s interesting
Melodix is more than just a clone of existing platforms:
It’s a learning lab for modern architectures and design patterns.
A chance to apply best practices in cybersecurity, streaming, and UX design.
A blend of clean design with powerful technical foundations.

🛠️ Tech Stack
Backend: Spring Boot
Frontend: React
Database: H2
Other tools: Maven, REST APIs, JWT authentication
